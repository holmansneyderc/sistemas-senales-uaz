# Nombre del Proyecto

Descripción breve del proyecto. Esta aplicación es una aplicación monolítica construida con Flask, Vue.js y desplegada en Heroku que realiza operaciones de convolución y Transformada de Fourier.

## Requisitos

- Python 3.8+
- Node.js 14+
- Heroku CLI
